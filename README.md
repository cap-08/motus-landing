# Motus — Humanoid Robotics & Rehabilitation Devices Landing Page

A single-page, fully responsive concept landing page for a deep-tech company
operating in two domains — **Humanoid Robotics** and **Rehabilitation Devices**
— presented with equal visual weight and tied together by a shared "motion
spine" visual motif.

## View it live
Once GitHub Pages is enabled on this repo (see below), the page will be at:
`https://<your-username>.github.io/<repo-name>/`

## Files
- `index.html` — the landing page (HTML/CSS/JS, no build step needed)
- `screenshot-desktop.png`, `screenshot-mobile.png` — full-page screenshots
- `Deliverables_and_AI_Usage.pdf` — submission doc with AI-usage disclosure

## Run locally
Just open `index.html` in any browser — no server or dependencies required.
